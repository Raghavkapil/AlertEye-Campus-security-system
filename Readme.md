# About Project
## This project have two parts (or perspective):
```
a) Safety-App: User (or student) can trigger an alert from their end which will notify the trusted contacts with user's name, current location and time.
```
```
b) Dashboard: User (or security personnel) can access and manage this dashboard which displays the live feed of the camera and determines in real time whether there's some abnormality or not and notify the guard with the same (if found).

```

### Tech Stack:
```
>App: React Native, Django, Firebase, Gemini Engine (along with API keys)

>Dashboard: React Native, Django, Firebase 

>For Model: YOLO v8(You Only Look Once), Gemini Engine
```

### Look for Readme.md file inside each folder to run the app and dashboard locally
